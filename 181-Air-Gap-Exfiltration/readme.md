## Description
A technique to steal data from **Air-Gapped** (isolated) computers that have no internet connection.
It encodes binary data into **high-frequency sound waves (Ultrasonic)** played through the computer speakers. These sounds are inaudible to humans but can be recorded and decoded by a nearby smartphone microphone.
