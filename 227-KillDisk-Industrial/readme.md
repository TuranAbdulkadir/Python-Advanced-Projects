# ☢️ Project 227: KillDisk Industrial
**Type:** Cyber Warfare / Wiper
**Tech:** Raw Disk Access

Based on the logic of industrial wipers (like Shamoon/KillDisk).
It targets specific sectors on the hard drive that store the File System structure (MBR, VBR, MFT Mirror). By corrupting these few kilobytes, the entire Terabyte drive becomes unreadable raw data.
