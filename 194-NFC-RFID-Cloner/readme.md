## Description
Interface tool for RFID cloning hardware (like Proxmark3 or Arduino RC522).
It simulates the process of:
1. Reading the **Unique Identifier (UID)** from a legitimate access card.
2. Writing that UID onto a blank card, effectively cloning the key.
